# Cybersecurity Metrics Dashboard - Power BI

Interactive Power BI dashboard for monitoring cybersecurity metrics, incident tracking, and threat intelligence.

### 🎯 Objective
Transform raw security logs into actionable business insights for SOC teams and IT managers.

### ✨ Features
- Failed login attempts trend
- Top attacking IPs and countries
- Vulnerability distribution
- Incident response time analysis
- Executive summary page

### 🛠️ Technologies
- Power BI Desktop
- DAX
- Power Query
- Data Visualization
